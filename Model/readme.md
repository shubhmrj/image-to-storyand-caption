# Model Overview

This directory contains two fine-tuned Shubham Raj trained Hugging Face models used for image captioning and story generation.

## Models

- **Story generation**
  - Model: `gpt2-rocstories-finetuned`
  - Hugging Face: https://huggingface.co/shubmrj/gpt2-rocstories-finetuned
  - Description: Generates short ROCStories-style narratives from text prompts.

- **Image captioning**
  - Model: `blip-coco-finetuned`
  - Hugging Face: https://huggingface.co/shubmrj/blip-coco-finetuned
  - Description: Produces descriptive captions for images.

## Usage

1. Use `gpt2-rocstories-finetuned` for story or narrative generation.
2. Use `blip-coco-finetuned` for image captioning tasks.

## Notes

- Both models are hosted under the Hugging Face user `shubmrj`.
- They can be used together in applications that combine visual captioning with story generation.
